# lilPick

Giveaway Winner Picker

Paste your comments or entries, pull out every @handle, and draw a fair random winner with crypto-grade randomness. Includes a copyable AI prompt to turn a messy comment thread into a clean entry list.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
